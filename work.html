Gradebook

In this project, I was given a basic HTML and CSS structure. I had to create a page capable of taking the data entered by the user and put it into a table without editing the HTML or CSS originally given. Added features include: auto-correct of subject if entered incorrectly or mispelled, a 'late' button which will take 10 points off any assignment if not 'on-time', running GPA for all subjects as well as individual subjects, and the ability to make changes to any entered information by clicking on the part you wish to change (do note for this feature it will update GPA and subject GPA accordingly as well).

Budget Application

This project creates a running budget for the user. This project came with a HTML and CSS design that I was to add JavaScript to make it functional with no edits to HTML or CSS files. This application comes with the ability to add either an income or an expense as well as track the ongoing total of the two. Note: this was a guided project. Added features outside the guided portion include: alerts user when they are over budget, proper commas will be added no matter how large the number, user can edit any portion of their entry by clicking on the value to edit.

The Pig Game

Created the classic Pig Game through manipulation of JavaScript. Game will allow user to roll a dice as many times as they like, adding their total to the current category. At any time they may stop and add the current score to their total score, ending their turn. If they roll a 1, the current score returns to 0 and their turn is ended. Set to two players and the first to 100 points wins.
